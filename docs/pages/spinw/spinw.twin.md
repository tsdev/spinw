### Sub fields
 
`rotc`
: Rotation matrices in the $$xyz$$ coordinate system for
  every twin, stored in a matrix with dimensions of $$[3\times
  3\times n_{twin}]$$.
 
`vol`
: Volume ratio of the different twins, stored in a
   row vector with $$n_{twin}$$ elements.
