### Syntax
  
`mem = sw_freemem`
  
### Description
  
`mem = sw_freemem` determines the available free memory (RAM). If the
function cannot determine the size of the free memory, it returns zero.
  
### Output Arguments
 
`mem`
: Size of free memory in bytes.
